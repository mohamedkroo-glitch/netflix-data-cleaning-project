This dataset has been cleaned and standardized with the following steps:

Handling Missing Values

3,655 empty cells were replaced with the value "Unknown".

Removing Duplicates

Duplicate rows were identified using conditional formatting and removed completely.

Date Standardization

The date_added column was reformatted to a consistent date format.

Column Name Normalization

All column names were converted to lowercase and spaces were removed for consistency.

Data Type Validation

Ensured that each column has the correct data type (numeric, text, or date).
